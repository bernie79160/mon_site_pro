Maîtriser Windows

Windows est le "chef d'orchestre" de votre ordinateur. Il vous permet de ranger vos documents comme dans un vrai bureau.

1. Le Bureau et la Souris

Le **Bureau** est l'écran que vous voyez au démarrage. C'est votre table de travail.

  • Le Clic Gauche : Pour sélectionner ou choisir quelque chose.
  • Le Clic Droit : Pour ouvrir un menu d'options (comme un menu de restaurant).
  • Le Double-Clic : Pour ouvrir un dossier ou un logiciel.

2. Le Bouton "Démarrer"

C'est la porte d'entrée de votre ordinateur (en bas à gauche). Si vous cherchez un logiciel ou si vous voulez éteindre l'ordinateur, c'est ici !

3. Les Fichiers et Dossiers

Pour ne pas tout mélanger, on utilise des dossiers :

Un Fichier : C'est une lettre, une photo ou une musique.

Un Dossier : C'est une chemise cartonnée où l'on range plusieurs fichiers.
